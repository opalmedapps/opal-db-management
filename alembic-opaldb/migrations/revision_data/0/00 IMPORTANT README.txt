The primary purpose of this folder "0 Drop Table Holding SQL" is to hold
any SQL scripts that contains "Drop Table" or "Drop Column".

Do not DROP TABLE or DROP COLUMN right away because we want to maintain backward compatability.

These scripts are placed here until it is deemed ready to be used.

Ready as in 
1. it is confirmed that all current release no longer uses the table or column.
2. it is confirmed that no new recores are being inserted into the table or column.
